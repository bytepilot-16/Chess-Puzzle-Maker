A lightweight, browser-based chess puzzle creator and database manager. This application allows users to set up custom board states, record solution paths, and save them to a local database for batch JSON exporting[cite: 1].

## ✨ Features

* **Creator Studio:** Drag-and-drop interface to build custom chess positions from scratch or standard starts[cite: 1].
* **Solution Recording:** Interactive board that validates legal moves (powered by `chess.js`) and records the exact solution sequence[cite: 1].
* **Local Database:** Saves generated puzzles locally to the browser, allowing users to build a collection over time[cite: 1].
* **Batch Export:** One-click export of the entire puzzle database into a structured JSON file[cite: 1].
* **Responsive UI:** Built with Tailwind CSS, featuring custom toast notifications and a dark-mode-first aesthetic[cite: 1].

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Tailwind CSS (via CDN)[cite: 1]
* **Logic & DOM Manipulation:** JavaScript, jQuery[cite: 1]
* **Chess Libraries:** `chessboard.js` (UI) and `chess.js` (move validation/logic)[cite: 1]

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to launch the studio. No build steps or local server required.
